## 📖 核心内容

### 1. 不定式的概念

在求极限时，经常遇到以下形式（称为**不定式**）：

- $\frac{0}{0}$ 型
- $\frac{\infty}{\infty}$ 型  
- $0 \cdot \infty$ 型
- $\infty - \infty$ 型
- $0^0$、$\infty^0$、$1^{\infty}$ 型

这些形式不能直接判断极限值，需要**化简变形**后再求极限。

今天我们主要学习处理 **$\frac{0}{0}$ 型** 的方法。

### 2. 因式分解法

**适用场景：** 分子分母都含有因式 $(x - a)$，代入后得 $\frac{0}{0}$

**方法：** 约去公因子，化简后再求极限

#### 例1：最简单的情况

计算：$\lim_{x \to 2} \frac{x^2 - 4}{x - 2}$

**解答：**

直接代入得 $\frac{0}{0}$，需要化简。

因式分解：
$$\lim_{x \to 2} \frac{x^2 - 4}{x - 2} = \lim_{x \to 2} \frac{(x-2)(x+2)}{x - 2}$$

约去 $(x-2)$（因为求极限时 $x \neq 2$）：
$$= \lim_{x \to 2} (x + 2) = 4$$

**关键：** 因为是求 $x \to 2$ 的极限，所以 $x \neq 2$，可以约去 $(x-2)$

#### 例2：三次公式

计算：$\lim_{x \to 1} \frac{x^3 - 1}{x - 1}$

**解答：**

利用立方差公式：$a^3 - b^3 = (a-b)(a^2 + ab + b^2)$

$$\lim_{x \to 1} \frac{x^3 - 1}{x - 1} = \lim_{x \to 1} \frac{(x-1)(x^2 + x + 1)}{x - 1}$$

$$= \lim_{x \to 1} (x^2 + x + 1) = 3$$

**常用公式：**
- $a^2 - b^2 = (a-b)(a+b)$
- $a^3 - b^3 = (a-b)(a^2 + ab + b^2)$
- $a^3 + b^3 = (a+b)(a^2 - ab + b^2)$
- $a^n - b^n = (a-b)(a^{n-1} + a^{n-2}b + \cdots + ab^{n-2} + b^{n-1})$

#### 例3：复杂的因式分解

计算：$\lim_{x \to -1} \frac{x^3 + 2x^2 - x - 2}{x^2 - 1}$

**解答：**

分子因式分解（用综合除法或试根法）：
$$x^3 + 2x^2 - x - 2 = (x+1)(x^2 + x - 2) = (x+1)(x+2)(x-1)$$

分母因式分解：
$$x^2 - 1 = (x+1)(x-1)$$

所以：
$$\lim_{x \to -1} \frac{(x+1)(x+2)(x-1)}{(x+1)(x-1)} = \lim_{x \to -1} (x+2) = 1$$

### 3. 有理化法

**适用场景：** 分子或分母含有根式，造成 $\frac{0}{0}$ 型

**方法：** 乘以共轭式，化去根号

#### 例4：分子有理化

计算：$\lim_{x \to 0} \frac{\sqrt{x+1} - 1}{x}$

**解答：**

分子分母同乘 $\sqrt{x+1} + 1$（共轭式）：

$$\lim_{x \to 0} \frac{\sqrt{x+1} - 1}{x} = \lim_{x \to 0} \frac{(\sqrt{x+1} - 1)(\sqrt{x+1} + 1)}{x(\sqrt{x+1} + 1)}$$

$$= \lim_{x \to 0} \frac{(x+1) - 1}{x(\sqrt{x+1} + 1)} = \lim_{x \to 0} \frac{x}{x(\sqrt{x+1} + 1)}$$

$$= \lim_{x \to 0} \frac{1}{\sqrt{x+1} + 1} = \frac{1}{2}$$

**技巧：** $(\sqrt{a} - \sqrt{b})(\sqrt{a} + \sqrt{b}) = a - b$

#### 例5：分母有理化

计算：$\lim_{x \to 0} \frac{x}{\sqrt{x+4} - 2}$

**解答：**

分子分母同乘 $\sqrt{x+4} + 2$：

$$\lim_{x \to 0} \frac{x(\sqrt{x+4} + 2)}{(\sqrt{x+4} - 2)(\sqrt{x+4} + 2)}$$

$$= \lim_{x \to 0} \frac{x(\sqrt{x+4} + 2)}{(x+4) - 4} = \lim_{x \to 0} \frac{x(\sqrt{x+4} + 2)}{x}$$

$$= \lim_{x \to 0} (\sqrt{x+4} + 2) = 4$$

#### 例6：双重有理化

计算：$\lim_{x \to 0} \frac{\sqrt{1+x} - \sqrt{1-x}}{x}$

**解答：**

分子分母同乘 $\sqrt{1+x} + \sqrt{1-x}$：

$$\lim_{x \to 0} \frac{(\sqrt{1+x} - \sqrt{1-x})(\sqrt{1+x} + \sqrt{1-x})}{x(\sqrt{1+x} + \sqrt{1-x})}$$

$$= \lim_{x \to 0} \frac{(1+x) - (1-x)}{x(\sqrt{1+x} + \sqrt{1-x})} = \lim_{x \to 0} \frac{2x}{x(\sqrt{1+x} + \sqrt{1-x})}$$

$$= \lim_{x \to 0} \frac{2}{\sqrt{1+x} + \sqrt{1-x}} = \frac{2}{2} = 1$$

### 4. 换元法

**适用场景：** 通过变量替换简化表达式

#### 例7：换元简化

计算：$\lim_{x \to 0} \frac{\sqrt{1+x} - \sqrt{1-x}}{x}$（用换元法）

**解答：**

令 $t = \sqrt{1+x}$，则 $x = t^2 - 1$，当 $x \to 0$ 时，$t \to 1$

类似地考虑 $\sqrt{1-x}$，这个例子换元较复杂，不如有理化法。

**注：** 换元法在某些特殊情况下有效，需要灵活运用。

### 5. 金融应用：债券收益率

债券的到期收益率r满足：
$$P = \frac{C}{1+r} + \frac{C}{(1+r)^2} + \cdots + \frac{C+F}{(1+r)^n}$$

其中P是价格，C是票息，F是面值。

当 $r \to 0$ 时（利率趋于0），价格趋于什么？

$$\lim_{r \to 0} P = \lim_{r \to 0} \left[\frac{C}{1+r} + \frac{C}{(1+r)^2} + \cdots + \frac{C+F}{(1+r)^n}\right]$$

$$= C + C + \cdots + (C + F) = nC + F$$

**经济意义：** 当无风险利率为0时，债券价格等于所有现金流的简单加总。

## 📊 典型例题总结

### 题型1：平方差公式

$$\lim_{x \to a} \frac{x^2 - a^2}{x - a} = \lim_{x \to a} (x + a) = 2a$$

### 题型2：立方差公式

$$\lim_{x \to a} \frac{x^3 - a^3}{x - a} = \lim_{x \to a} (x^2 + ax + a^2) = 3a^2$$

### 题型3：根式有理化

$$\lim_{x \to 0} \frac{\sqrt{x+a} - \sqrt{a}}{x} = \frac{1}{2\sqrt{a}}$$

## 💡 重要提示

1. **方法选择**
   - 看到 $x^2 - a^2$ 等 → 因式分解
   - 看到 $\sqrt{x+a} - \sqrt{b}$ → 有理化
   - 形式复杂 → 考虑换元

2. **常见错误**
   - ❌ 直接约去为0的因子
   - ✅ 先因式分解，再约去公因子

3. **解题步骤**
   1. 代入检验是否为不定式
   2. 选择合适的方法化简
   3. 再次代入求值

4. **技巧积累**
   - 熟记因式分解公式
   - 掌握共轭式的运用
   - 多练习才能熟能生巧
