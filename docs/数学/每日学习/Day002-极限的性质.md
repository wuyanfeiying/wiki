## 📖 核心内容

### 1. 极限的唯一性

**定理：** 如果 $\lim_{x \to a} f(x)$ 存在，则极限值是唯一的。

**意义：** 一个函数在某点的极限不可能既等于A又等于B（A≠B）。

这个性质看似显然，但很重要：它保证了极限作为一个数学概念的合理性。

### 2. 极限的局部有界性

**定理：** 如果 $\lim_{x \to a} f(x) = A$，则存在 $\delta > 0$，使得当 $0 < |x - a| < \delta$ 时，$|f(x)|$ 有界。

**直观理解：** 如果函数在某点有极限，那么在这点附近函数不会"爆炸"（无界）。

**例子：**
- $f(x) = x^2$ 在 $x \to 2$ 时有极限，在 x = 2 附近有界
- $f(x) = \frac{1}{x}$ 在 $x \to 0$ 时极限不存在，在 x = 0 附近无界

### 3. 极限的保号性

**定理：** 如果 $\lim_{x \to a} f(x) = A > 0$，则存在 $\delta > 0$，使得当 $0 < |x - a| < \delta$ 时，$f(x) > 0$。

**直观理解：** 
- 如果极限是正数，那么在靠近该点的地方，函数值也是正数
- 类似地，如果极限是负数，函数值也是负数

**应用：** 判断函数的符号，这在证明不等式时很有用。

### 4. 极限的四则运算法则 ⭐⭐⭐

这是**最重要**的性质！

设 $\lim_{x \to a} f(x) = A$，$\lim_{x \to a} g(x) = B$，则：

#### (1) 加法法则
$$\lim_{x \to a} [f(x) + g(x)] = A + B$$

#### (2) 减法法则
$$\lim_{x \to a} [f(x) - g(x)] = A - B$$

#### (3) 乘法法则
$$\lim_{x \to a} [f(x) \cdot g(x)] = A \cdot B$$

**推论：** 常数可以提出来
$$\lim_{x \to a} [c \cdot f(x)] = c \cdot A$$

#### (4) 除法法则
$$\lim_{x \to a} \frac{f(x)}{g(x)} = \frac{A}{B} \quad (B \neq 0)$$

**注意：** 分母的极限不能为0！

#### (5) 幂次法则
$$\lim_{x \to a} [f(x)]^n = A^n$$

### 5. 夹逼准则（夹逼定理）⭐⭐⭐

**定理：** 如果在 a 的某个去心邻域内，有：
$$g(x) \leq f(x) \leq h(x)$$

且 $\lim_{x \to a} g(x) = \lim_{x \to a} h(x) = A$

则：$\lim_{x \to a} f(x) = A$

**直观理解：** 如果f(x)被夹在g(x)和h(x)之间，而g和h都趋于同一个值A，那么f也必然趋于A。

**应用场景：** 当直接计算困难时，通过找上下界来求极限。

## 📊 典型例题

### 例题1：利用四则运算求极限

计算：$\lim_{x \to 2} \frac{x^2 + 3x - 1}{x + 1}$

**解答：**

因为分子、分母的极限都存在且分母极限不为0，可以直接代入：

$$\lim_{x \to 2} \frac{x^2 + 3x - 1}{x + 1} = \frac{\lim_{x \to 2}(x^2 + 3x - 1)}{\lim_{x \to 2}(x + 1)} = \frac{4 + 6 - 1}{2 + 1} = \frac{9}{3} = 3$$

**注意：** 这种"直接代入"的方法有前提条件：
1. 分子、分母极限都存在
2. 分母极限不为0
3. 没有出现 $\frac{0}{0}$、$\frac{\infty}{\infty}$ 等不定式

### 例题2：多项式函数的极限

计算：$\lim_{n \to \infty} \frac{3n^2 + 2n + 1}{2n^2 - n + 5}$

**解答：**

分子分母同时除以 $n^2$（最高次项）：

$$\lim_{n \to \infty} \frac{3n^2 + 2n + 1}{2n^2 - n + 5} = \lim_{n \to \infty} \frac{3 + \frac{2}{n} + \frac{1}{n^2}}{2 - \frac{1}{n} + \frac{5}{n^2}}$$

因为 $\lim_{n \to \infty} \frac{1}{n} = 0$，$\lim_{n \to \infty} \frac{1}{n^2} = 0$

所以：$$= \frac{3 + 0 + 0}{2 - 0 + 0} = \frac{3}{2}$$

**规律：** 
- 最高次幂相同：极限 = 最高次项系数之比
- 分子次数高：极限 = ∞
- 分母次数高：极限 = 0

### 例题3：夹逼准则的应用

计算：$\lim_{n \to \infty} \frac{\sin n}{n}$

**解答：**

我们知道：$-1 \leq \sin n \leq 1$

两边同时除以 n（n > 0）：
$$-\frac{1}{n} \leq \frac{\sin n}{n} \leq \frac{1}{n}$$

因为：
$$\lim_{n \to \infty} \left(-\frac{1}{n}\right) = 0, \quad \lim_{n \to \infty} \frac{1}{n} = 0$$

由夹逼准则：
$$\lim_{n \to \infty} \frac{\sin n}{n} = 0$$

### 例题4：金融应用 - 复利计算

投资本金 P = 10000元，年利率 r = 6%，投资n年。

(1) 如果按年复利，n年后的本利和为多少？  
(2) 如果按月复利，n年后的本利和为多少？  
(3) 如果连续复利，n年后的本利和为多少？

**解答：**

(1) 年复利：$A = P(1 + r)^n = 10000 \times 1.06^n$

(2) 月复利：$A = P\left(1 + \frac{r}{12}\right)^{12n} = 10000 \times \left(1 + \frac{0.06}{12}\right)^{12n}$

(3) 连续复利：
$$A = \lim_{m \to \infty} P\left(1 + \frac{r}{m}\right)^{mn} = P e^{rn} = 10000 \times e^{0.06n}$$

当 n = 10 年时：
- 年复利：$10000 \times 1.06^{10} \approx 17908$ 元
- 月复利：$10000 \times 1.005^{120} \approx 18194$ 元  
- 连续复利：$10000 \times e^{0.6} \approx 18221$ 元

**结论：** 计息频率越高，本利和越大，但增加有限。

## 💡 重要提示

1. **运算法则的前提**
   - 各个极限必须存在
   - 除法中分母极限不能为0

2. **常见错误**
   - ❌ $\lim \frac{f(x)}{g(x)} = \frac{\lim f(x)}{\lim g(x)}$（当分母为0时不成立）
   - ✅ 必须先确保分母极限不为0

3. **实用技巧**
   - 对于多项式比值：抓最高次项
   - 有界函数乘以无穷小：用夹逼准则
   - 不定式（$\frac{0}{0}$等）：需要特殊方法（后续学习）

4. **记忆要点**
   - 四则运算法则是基础，必须熟练
   - 夹逼准则是重要工具，要会用
