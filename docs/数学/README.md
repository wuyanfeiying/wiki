## 📖 三个阶段

### 阶段一：微积分（1-80天）
- 极限与连续
- 导数与微分
- 积分
- 微分方程
- 多元微积分

### 阶段二：线性代数（81-140天）
- 矩阵与行列式
- 向量空间
- 线性方程组
- 特征值与特征向量
- 矩阵分解

### 阶段三：概率统计（141-180天）
- 概率基础
- 随机变量与分布
- 数理统计

## 💡 学习建议

1. **理解优于记忆**：不要死记公式
2. **多做练习**：数学需要大量练习
3. **联系金融**：每个知识点都有金融应用案例
4. **坚持到底**：180天后你会感谢现在努力的自己
