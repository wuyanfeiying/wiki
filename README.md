## 📝 使用指南

### 文档编写

1. 使用提供的[文档模板](templates/)创建新笔记
2. 文档使用Markdown格式编写
3. 图片等资源文件放在 `resources/` 目录

### 文档模板

- [知识点模板](templates/知识点模板.md) - 用于整理系统性知识点
- [学习笔记模板](templates/学习笔记模板.md) - 用于记录学习过程

### 命名规范

- 文件名使用中文，简洁明确
- 避免特殊字符，使用中划线分隔
- 示例：`期权定价-Black-Scholes模型.md`
